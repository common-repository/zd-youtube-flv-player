== Description ==

Display FLV and YouTube Video in your blog in a Custom Player.

= Usage =

[zdvideo]url[/zdvideo]

'url': This is the URL of either a youtube video or flv video file. Example - http://in.youtube.com/watch?v=7HdJgf3yXM0 or http://www.yourdomain.com/videos/myvideo.flv

Attributes:-
<ul>
<li>width - Default is "425". This is the width of the player including border.</li>
<li>height - Default is "349". This is the height of the player including border.</li>
<li>align - Alignment of the Flash Player. Default is "center". Available Options: "left"/"right"/"center"</li>
<li>border - Show/Hide Flash Player Border. Default "yes". Available Options: "yes"/"no"</li>
<li>fullscreen - Show/Hide full screen button. Default "yes". Available Options: "yes"/"no"</li>
<li>theme - Name of the theme you want to use for the Flash Player. Default "dark".</li>
<li>autoplay - Set video auto play On/off. Default "no". Available Options: "yes"/"no"</li>
</ul>

= Themes =

<ol>
<li>Dark</li>
<li>Glossy</li>
<li>Gray</li>
<li>Simple 1</li>
<li>Simple 2</li>
</ol>

This plugin uses SWFObject 2.1 to embed the flash player. So, the xHTML generated is W3C Valid.


== Installation ==

1. Upload the full directory into your 'wp-content/plugins' directory
2. Activate the plugin at the plugin administration page
3. Use [zdvideo]url[/zdvideo] shortcode in your post or page. Use the attributes as per your requirements.

== Frequently Asked Questions ==

= Minimum PHP Version =

PHP 5.1.x

No. This plugin only supports youTube videos and FLV videos.
